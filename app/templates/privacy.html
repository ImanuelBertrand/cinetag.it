2. Collection and Storage of Personal Data and the Type and Purpose of Their Use

a) Server Log Files
When you access our website, information is automatically collected by the web server and stored in server log files. The data collected includes:

  • IP address
  • Access time (timestamp)
  • URL of the accessed page
  • User-Agent (browser type and version)

Purpose of Processing: This data is used exclusively for administrative and security-related purposes, such as for error analysis and troubleshooting as part of website development and improvement.
Storage Duration: Server log files are stored for a maximum of 14 days and are automatically deleted afterward unless security incidents require a longer storage period.

b) User Account and Email Address
For using the account functions on our website, you may optionally create a user account. In this process, we collect:

  • Your email address
  • A password (stored in encrypted form)

Purpose of Processing: The email address is used for account management (e.g., login, “forgot password”). It is not used for any other purposes, such as marketing.
Optional email notifications for movie release dates can be activated or deactivated by the user within their account.
Storage Duration: The data is stored as long as the user has an account with us. Users can delete their accounts and data independently.

3. Data Subject Rights

As a user, you have the following rights regarding your personal data:

  • Right to access the data we store about you.
  • Right to correct incorrect data.
  • Right to deletion ("right to be forgotten") if no legal retention requirements exist.
  • Right to restrict processing if the accuracy of the data is contested, the processing is unlawful, or the data is no longer needed for the purpose of processing.
  • Right to data portability.

To exercise these rights or if you have any other questions about data privacy, you can contact us at the email address provided above.

4. External Services

Our website does not use any external services, tracking tools, web analytics, third-party fonts, CDNs, or similar tools. No data is transmitted to third parties.

5. Data Security

We use technical and organizational security measures to protect your personal data against loss, manipulation, or unauthorized access. Data transmission between your browser and our server is encrypted via TLS transport encryption to ensure confidentiality and data integrity. Passwords are stored in encrypted form using the bcrypt algorithm, which ensures secure and resilient storage against unauthorized access. Our security measures are regularly reviewed and enhanced in line with technological advancements.
